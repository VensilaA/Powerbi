Sales Analysis Dashboard (Power BI Project)
📌 Overview

This project is an interactive Sales Analysis Dashboard created using Microsoft Power BI.
It helps analyze business performance by visualizing sales data across different regions, products, and customer segments.

🎯 Objectives
Monitor overall sales performance
Analyze profit and quantity metrics
Identify top-performing products
Compare regional sales performance
Understand customer segment contribution
Tools & Technologies
Microsoft Power BI Desktop
CSV Dataset (Superstore Data)

📂 Dataset Details
The dataset used in this project contains:
Order Date
Product Name
Category
Region
Customer Segment
Sales
➕ Additional Calculated Columns:
Profit = Sales * 0.2
Quantity = 1
Target = Sales * 0.9
📊 Dashboard Structure
🟦 Page 1: Sales Overview Dashboard

This page provides a summary of overall performance.

Includes:
KPI Cards (Total Sales, Total Profit, Total Quantity)
Sales Trend Over Time (Line Chart)
Sales by Region (Column Chart)
Category Contribution (Pie Chart)
Slicers (Date, Region, Category, Segment)
Page 2: Sales Detailed Analysis

This page focuses on deeper insights.

Includes:
Top Selling Products (Bar Chart)
Salesperson Performance (Column Chart)
Sales vs Profit Analysis (Scatter Chart)
Customer Segment Distribution (Donut Chart)

🔍 Key Insights
Sales trends vary across regions and time
Certain products contribute more to revenue
Customer segments influence sales distribution
Profit is directly proportional to sales
